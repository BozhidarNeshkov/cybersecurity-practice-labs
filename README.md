# Cybersecurity Practice Labs

This repository contains my hands-on cybersecurity exercises, writeups, and research notes.  
I use it to document my progress as I build practical skills across both defensive and offensive security.

The work here is based on:
- PortSwigger labs
- TryHackMe rooms
- SoftUni cybersecurity modules
- Personal research and experimentation

The purpose of this repository is to document my learning process, demonstrate practical skills, and build a structured portfolio.

Each lab includes:
- A clear explanation of the vulnerability or concept
- Steps taken to analyze or exploit it
- Payloads, requests, and methodology
- Screenshots and evidence
- Key takeaways and lessons learned

This repository will continue to grow as I complete more labs and expand my knowledge in web security, network security, and penetration testing fundamentals.
